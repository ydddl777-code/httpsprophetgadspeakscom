# Hand in Hand

*As Enoch walked and talked with You.*

A family devotional companion app — KJV Scripture, counsel, and wisdom for every age (child, teen, parent, elder).

**A Remnant Seed LLC product.** Deployed at [theprophetgad.com](https://theprophetgad.com).

## Stack

- Vite + React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (auth, database, edge functions)
- React Router
- ElevenLabs TTS for Prophet Gad voice

## Develop locally

Requires Node.js & npm.

```sh
git clone https://github.com/ydddl777-code/httpsprophetgadspeakscom.git
cd httpsprophetgadspeakscom
npm install
npm run dev
```

## Scripts

- `npm run dev` — local dev server
- `npm run build` — production build
- `npm run lint` — lint
- `npm run test` — run vitest

## Deployment

Target: Horizon VPS at theprophetgad.com.
